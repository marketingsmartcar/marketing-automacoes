## Como Usar a Automação Canva

### Para gerar 1 arte (1 página, 1 unidade):
1. `start-editing-transaction` com `design_id: DAHGdMRISHc`
2. `perform-editing-operations` com `find_and_replace_text` nos campos da página desejada
3. `commit-editing-transaction`
4. `export-design` como PNG (1080×1350)

### Para gerar para TODA a rede (múltiplas unidades):
Repetir o processo acima para cada cidade, trocando apenas os dados locais.
Cada execução cria uma variante exportada por unidade.

### Campos trocados automaticamente por página:

**Página 1 — Serviço:**
| Placeholder | Substituído por |
|-------------|----------------|
| Texto headline | Nome do serviço em maiúsculas |
| Texto serviço | Descrição completa do serviço |
| Texto preço | Preço informado (ex: "R$179") |
